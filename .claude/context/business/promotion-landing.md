# 프로모션 랜딩

## 목적

숙박 업체 등 파트너사에게 YesTravel의 인플루언서 마케팅 서비스를 소개하고, 제휴 문의를 접수받아 비즈니스 파트너십을 확대한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 서비스 소개 | 인플루언서 마케팅 서비스의 특장점 안내 (원고료, 방문촬영, 2차 활용) | 파트너사가 서비스 이해 |
| 포트폴리오 전시 | 기존 진행 사례(공구/방문 피드) 소개 | 실제 사례를 통한 신뢰 구축 |
| 인플루언서 소개 | 소속 인플루언서 목록 및 플랫폼 정보 노출 | 협업 가능한 인플루언서 확인 |
| 제휴 문의 접수 | 브랜드명/연락처 입력 폼 (데스크톱 인라인, 모바일 바텀시트 모달) | 간편한 상담 신청 |
| FAQ | 자주 묻는 질문 아코디언 | 궁금증 자가 해소 |

## 사용자 흐름

1. 파트너사 담당자가 프로모션 페이지에 접속한다
2. 서비스 소개, 포트폴리오, 인플루언서 정보를 확인한다
3. 하단 고정 바에서 브랜드명과 연락처를 입력하여 무료 상담을 신청한다
4. 모바일에서는 "제휴 문의하기" 버튼 탭 → 바텀시트 모달에서 입력한다
