# 프로모션 랜딩

## 목적

브랜드 담당자에게 인플루언서 마케팅 서비스의 가치를 전달하고, 제휴 문의로 전환시키는 B2B 마케팅 랜딩 페이지. 서비스 실적, 콘텐츠 레퍼런스, 인플루언서 풀을 보여주어 신뢰를 구축한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 서비스 소개 | 인플루언서 커머스 & 마케팅 플랫폼 소개 | 서비스 핵심 가치를 한눈에 파악 |
| 실적 증명 | 600+ 브랜드, 2,500+ 캠페인 파트너사 로고 노출 | 서비스 신뢰도 판단 |
| 콘텐츠 레퍼런스 | 정보형/방문형 인스타 피드 포트폴리오 | 콘텐츠 퀄리티 확인 |
| 서비스 차별점 | 단가 50만원~, 방문촬영, 2차 활용 특장점 | 경쟁 서비스 대비 장점 비교 |
| 인플루언서 소개 | 15인 인플루언서 프로필 (플랫폼별 팔로워 수) | 인플루언서 풀 규모와 품질 확인 |
| FAQ | 카테고리, 비용, 절차, 소재 활용 질문 답변 | 구매 전 의문점 해소 |
| 제휴 문의 | 하단 고정 CTA + 문의 폼 (모바일: 바텀시트) | 서비스 도입 문의 |
| 제안서 다운로드 | 헤더/히어로 CTA 버튼 | 상세 서비스 제안서 확보 |

## 사용자 흐름

1. 브랜드 담당자가 `/promo` 페이지 방문 → 히어로 배너에서 서비스 개요 확인
2. 스크롤하며 파트너 실적, 포트폴리오, 특장점 확인 → 서비스 신뢰도 형성
3. 인플루언서 프로필 확인 → 인플루언서 풀 품질 검증
4. FAQ로 의문점 해소 → 제휴 문의 폼 작성 또는 제안서 다운로드

## 전환 포인트 (CTA)

| CTA | 위치 | 동작 |
|-----|------|------|
| 제안서 다운로드 | 헤더, 히어로 | 외부 링크 (제안서 PDF) |
| 레퍼런스 보기 | 히어로 | 포트폴리오 섹션으로 스크롤 |
| 제휴 문의 | 하단 고정 바 | 모바일: 바텀시트 모달 / 데스크톱: 인라인 폼 |
