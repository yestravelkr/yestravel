# 인플루언서 관리

## 목적

인플루언서 파트너를 등록하고 관리하여, 각 인플루언서의 전용 커머스 페이지(`/i/{slug}`)를 통해 여행 상품을 판매할 수 있도록 한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 인플루언서 목록 조회 | 등록된 인플루언서 검색 및 조회 | 관리자가 인플루언서를 빠르게 찾아 관리 |
| 인플루언서 등록 | 기본 정보, 소셜미디어, 사업자 정보 입력 | 새로운 인플루언서 파트너 온보딩 |
| 인플루언서 수정 | 기존 정보 조회 및 수정 | 파트너 정보 변경 시 업데이트 |
| 커머스 페이지 연동 | slug 기반 전용 페이지 자동 생성 | 인플루언서별 개인화된 판매 채널 |

## 사용자 흐름

1. 관리자가 인플루언서 목록 페이지에서 이름/이메일/연락처로 검색
2. 새 인플루언서 등록 시 기본 정보, 소셜미디어(최소 1개), 사업자 정보 입력
3. 등록된 인플루언서의 slug가 커머스 페이지 URL에 사용됨
4. 기존 인플루언서 클릭 시 상세 정보 조회 및 수정 가능

## 도메인 개념

| 개념 | 설명 |
|------|------|
| 인플루언서 | SNS 채널을 보유한 여행 상품 판매 파트너 |
| Slug | 인플루언서 전용 커머스 페이지 URL 식별자 (영문 소문자, 숫자, 하이픈, 언더스코어) |
| 소셜미디어 | 인플루언서의 SNS 플랫폼 정보 (인스타그램, 유튜브, 틱톡 등) |
| 사업자 정보 | 정산을 위한 개인/법인 사업자 정보 |
| 정산 계좌 | 판매 수익 정산을 위한 은행 계좌 정보 |
