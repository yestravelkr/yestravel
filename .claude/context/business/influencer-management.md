# 인플루언서 관리

## 목적

여행 상품을 홍보하는 인플루언서 파트너를 등록하고 관리한다. 인플루언서는 캠페인에 참여하여 호텔/상품을 홍보하고, 이에 대한 정산을 받는다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 인플루언서 등록 | 기본정보, 소셜미디어, 사업자정보, 정산계좌 입력 | 관리자가 새 인플루언서 파트너 등록 |
| 인플루언서 수정 | 등록된 정보 수정 | 파트너 정보 변경 시 업데이트 |
| 인플루언서 조회 | 목록 검색 및 상세 확인 (소셜미디어 텍스트 표시) | 파트너 정보 확인 및 관리 |
| 이메일 필수화 | 이메일 주소 필수 입력 | 파트너 연락 수단 보장 |

## 사용자 흐름

1. 관리자가 인플루언서 목록 페이지에서 파트너 현황 확인
2. 신규 인플루언서 등록 시 기본정보(이름, 이메일, 전화번호) + 소셜미디어 계정 입력
3. 사업자 정보와 정산 계좌 정보를 추가로 입력
4. 상세 페이지에서 소셜미디어 정보를 플랫폼명+URL 텍스트로 확인
5. 수정 모드 전환 시 소셜미디어 추가/삭제/변경 가능
6. 등록된 인플루언서를 캠페인에 연결하여 상품 홍보 진행
7. 캠페인 완료 후 정산 처리

## 도메인 규칙

- 인플루언서 이름과 샵 URL(slug)은 고유해야 한다
- 최소 1개 이상의 소셜미디어 계정이 필요하다
- 이메일은 필수 입력값이다
- 지원 소셜미디어: Instagram, TikTok, YouTube, Facebook, Twitter, 기타
