# 상품 탐색

## 목적

고객이 여행 상품(호텔, E-티켓, 배송상품)의 상세 정보를 확인하고, 판매자 신뢰도를 검증하며, 같은 캠페인 내 다른 상품을 발견할 수 있도록 한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 상품 상세 보기 | 상품 타입별 상세 정보 렌더링 | 호텔/E-티켓/배송상품 각각에 맞는 상세 정보 확인 |
| 판매정보 조회 | 판매자 사업자 정보 표시 (업체명, 대표, 주소, 사업자등록번호) | 판매자 신뢰도 확인 및 문의처 파악 |
| 문의하기 | 판매자에게 연락할 수 있는 CTA 제공 | 상품에 대한 질문이나 요청 전달 |
| 다른 상품 보기 | 동일 캠페인 내 다른 상품 카드 목록 | 관련 상품 발견 및 비교 구매 유도 |

## 사용자 흐름

1. 고객이 상품 링크(`/sale/{saleId}`)로 진입하면 상품 상세 정보 표시
2. 탭 전환으로 판매정보 확인: 판매자 사업자 정보 + 문의하기 버튼
3. 탭 전환으로 다른 상품 보기: 같은 캠페인의 다른 상품 카드 목록에서 선택
