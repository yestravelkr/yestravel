# 관리자 관리

## 목적

백오피스 시스템의 접근 권한을 관리하여 역할별로 적절한 기능을 제공한다. 최고 관리자(슈퍼 관리자)가 일반 관리자와 파트너 관리자를 등록/삭제하고 비밀번호를 재설정할 수 있다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 관리자 목록 | 등록된 관리자 조회 및 검색 | 이름/이메일/연락처로 관리자 검색 |
| 관리자 등록 | 새 관리자 계정 생성 | 이름, 이메일, 연락처, 역할 지정 |
| 비밀번호 재설정 | 관리자 비밀번호 변경 | 슈퍼 관리자만 다른 관리자 비밀번호 초기화 |
| 관리자 삭제 | 관리자 계정 제거 | 슈퍼 관리자만 삭제 가능, 복구 불가 |
| 권한 판별 | JWT 토큰 기반 역할 확인 | 역할에 따라 관리 기능 접근 제한 |

## 사용자 흐름

1. 슈퍼 관리자가 관리자 목록 페이지 접근 → 전체 관리자 목록 확인
2. 검색어 입력 → 이름/이메일/연락처 기준 실시간 필터링
3. 관리자 행 클릭 → 상세 페이지로 이동
4. 액션 메뉴에서 비밀번호 재설정 → 새 비밀번호 입력 모달 → 변경 완료
5. 액션 메뉴에서 삭제 → 확인 모달 → 삭제 완료

## 권한 체계

| 역할 | 설명 | 관리 권한 |
|------|------|----------|
| 최고 관리자 | 시스템 전체 관리 | 관리자 등록/삭제, 비밀번호 재설정 |
| 관리자 | 일반 백오피스 운영 | 목록 조회만 가능 |
| 대표 관리자 | 파트너사 대표 | 파트너 영역 관리 |
| 관리자(파트너) | 파트너사 스태프 | 파트너 영역 제한적 접근 |
