# 호텔 취소 수수료 정책

## 목적

호텔 예약 취소 시 체크인까지 남은 일수에 따라 차등 수수료를 적용하여, 고객에게 명확한 환불 기준을 제공하고 호텔 파트너의 수익을 보호한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 수수료 정책 설정 | 체크인 잔여일별 수수료율(%) 배열 | 관리자가 호텔별 취소 정책 커스터마이징 |
| 다단계 수수료 | 여러 구간 설정 가능 | 30일전 0%, 7일전 50%, 당일 100% 등 |
| 기본값 무료 취소 | 정책 미설정 시 수수료 없음 | 별도 설정 없이도 동작 |

## 사용자 흐름

1. 관리자가 백오피스에서 호텔 상품 생성/수정 시 취소 수수료 정책을 입력
2. 체크인 잔여일(daysBeforeCheckIn)과 수수료율(feePercentage, 0-100%)을 배열로 설정
3. 정책 미설정 시 기본값 빈 배열([])로 무료 취소 적용

## 비즈니스 규칙

| 규칙 | 설명 |
|------|------|
| 수수료율 범위 | 0% ~ 100% |
| 잔여일 기준 | 0일 이상의 정수 |
| 기본 정책 | 빈 배열 = 무료 취소 |
| 저장 형식 | JSONB 배열 |
