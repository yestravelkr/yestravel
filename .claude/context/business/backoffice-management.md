# 백오피스 관리

## 목적

관리자가 여행 상품, 주문, 파트너 설정 등 핵심 업무를 효율적으로 수행할 수 있는 통합 관리 인터페이스를 제공한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 사이드바 네비게이션 | 그룹별 메뉴로 주요 기능에 빠르게 접근 | 상품/주문/설정 영역을 한눈에 파악하고 이동 |
| 사용자 정보 표시 | 로그인한 관리자 이메일 표시 | 현재 로그인 계정 확인 |
| 로그아웃 | 세션 종료 후 로그인 페이지 이동 | 안전한 세션 관리 |

## 관리 영역

| 영역 | 포함 기능 | 대상 사용자 |
|------|----------|------------|
| 상품관리 | 품목/상품 등록, 캠페인 관리 | 상품 담당자 |
| 주문관리 | 주문 조회, 정산 처리 | 운영 담당자 |
| 설정 | 브랜드/인플루언서/관리자 계정 관리 | 시스템 관리자 |

## 사용자 흐름

1. 관리자가 로그인 → 인증 후 사이드바가 포함된 레이아웃 진입
2. 사이드바에서 원하는 메뉴 클릭 → 해당 관리 페이지로 이동
3. 작업 완료 후 로그아웃 버튼으로 세션 종료
