# 캠페인 인플루언서 수수료

## 목적

캠페인에 참여하는 인플루언서별로 수수료와 판매 조건을 설정하여, 인플루언서 마케팅의 비용을 관리하고 상품별 커미션을 세밀하게 제어한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 인플루언서 고정 수수료 | 캠페인 참여 대가로 인플루언서에게 지급하는 고정 금액 | 관리자가 인플루언서별 수수료를 설정 |
| 참여 기간 설정 | 캠페인 기본 기간 또는 인플루언서별 커스텀 기간 선택 | 인플루언서마다 다른 기간 운영 가능 |
| 상품 판매 상태 | 인플루언서별로 상품의 노출/숨김/품절 상태 제어 | 특정 인플루언서에게만 특정 상품 노출 |
| 커스텀 커미션 | 호텔 옵션별 날짜에 따른 세밀한 커미션 설정 | 성수기/비수기 등 날짜별 차등 커미션 |
| 엑셀 일괄 처리 | 날짜별 커미션을 엑셀로 다운로드/업로드 | 대량 커미션 데이터를 엑셀로 편리하게 관리 |

## 사용자 흐름

### 인플루언서 판매설정

1. 관리자가 캠페인 상세에서 인플루언서 목록의 [판매설정] 클릭
2. 캠페인 설정 탭에서 참여 기간과 고정 수수료 설정
3. 상품 판매 설정 탭에서 상품별 노출 상태와 커스텀 커미션 여부 설정
4. 커스텀 커미션 활성화 시 날짜별 커미션 테이블 표시 및 엑셀 처리 가능

### 커미션 엑셀 처리

1. 관리자가 [엑셀 처리] 클릭 → 다운로드 탭에서 현재 커미션 데이터가 포함된 엑셀 다운로드
2. 엑셀 파일에서 호텔 옵션별 날짜 커미션 수정
3. 업로드 탭에서 수정된 엑셀 업로드 → 유효성 검증 → 미리보기 확인
4. [적용] 클릭으로 커미션 데이터 반영

### 수수료 구조

```
캠페인 인플루언서
├── 고정 수수료: 캠페인 참여 대가 (없음 또는 커스텀 금액)
├── 참여 기간: 캠페인 기본 기간 또는 커스텀 기간
└── 상품별 설정
    ├── 판매 상태: 노출/숨김/품절
    ├── 커스텀 커미션 사용 여부
    └── 호텔 옵션별 날짜 커미션
        └── {"2025-01-15": 5000, "2025-01-16": 7000, ...}
```
