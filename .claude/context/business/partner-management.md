# 파트너 관리

## 목적

브랜드 파트너와 인플루언서 파트너가 각자의 전용 포털에 로그인하여 서비스를 관리할 수 있게 한다. 기존 백오피스(관리자 전용)와 분리된 파트너 전용 인터페이스를 제공하여, 파트너가 자율적으로 정보를 확인하고 관리할 수 있는 환경을 구축한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 파트너 유형 선택 | 브랜드/인플루언서 중 선택하여 진입 | 본인에게 맞는 포털로 자연스럽게 이동 |
| 파트너 로그인 | 이메일/비밀번호로 인증 | 파트너 전용 계정으로 안전하게 접속 |
| 자동 세션 유지 | 토큰 갱신으로 재로그인 최소화 | 끊김 없는 사용 경험 |
| 브랜드 대시보드 | 브랜드 파트너 전용 관리 화면 | 브랜드 관련 정보 한눈에 확인 |
| 인플루언서 대시보드 | 인플루언서 파트너 전용 관리 화면 | 인플루언서 활동 관련 정보 확인 |
| 관리자 관리 | 파트너 스태프 추가/역할 변경/비밀번호 재설정/제거 | 파트너 조직 내 사용자 권한 관리 |
| 역할 제한 | 파트너 관리자에게 파트너 역할만 부여 가능 (ADMIN 역할 차단) | 권한 분리를 통한 보안 강화 |
| 호텔 주문 관리 | 파트너가 직접 호텔 주문 목록 조회 및 상세 확인 | 주문 현황을 실시간으로 파악 |
| 주문 권한 제어 | 파트너는 읽기 + 제한된 액션만 가능 (백오피스 대비 축소) | 안전한 운영 범위 내에서 자율 관리 |

## 사용자 흐름

1. 파트너가 포털 접속 → 브랜드/인플루언서 유형 선택
2. 선택한 유형에 맞는 로그인 페이지로 이동 → 이메일/비밀번호 입력
3. 로그인 성공 → 해당 유형의 대시보드로 자동 이동
4. 대시보드에서 사이드바 내비게이션을 통해 세부 기능 접근
5. 주문 관리 → 호텔 주문 목록 조회 → 주문 상세 확인 (제한된 액션)
6. 세션 만료 시 자동 토큰 갱신 → 실패 시 로그인 페이지로 안내

## 파트너 유형

| 유형 | 설명 | 주요 기능 (예정) |
|------|------|-----------------|
| 브랜드 파트너 | 여행 상품을 제공하는 브랜드 사업자 | 브랜드 관리, 상품 등록, 캠페인 참여, 주문 관리 |
| 인플루언서 파트너 | 여행 상품을 홍보하는 인플루언서 | 캠페인 참여, 정산 확인, 활동 관리, 주문 관리 |
