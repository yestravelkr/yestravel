# 캠페인 관리

## 목적

백오피스 관리자가 마케팅 캠페인을 생성하고, 인플루언서를 배정하며, 캠페인별/상품별 성과(주문 수, 매출)를 모니터링할 수 있도록 한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 캠페인 CRUD | 캠페인 생성, 수정, 삭제 | 마케팅 캠페인 라이프사이클 관리 |
| 캠페인 리스트 조회 | 캠페인별 목록 + 필터/페이지네이션 | 전체 캠페인 현황 파악 |
| 상품별 리스트 조회 | 상품 단위로 캠페인 성과 조회 | 상품별 마케팅 효과 비교 |
| 인플루언서 배정 | 캠페인에 인플루언서 연결 + 수수료 설정 | 인플루언서별 맞춤 조건 관리 |
| 성과 통계 | 주문 수, 매출 자동 집계 | 캠페인 ROI 판단 |

## 사용자 흐름

### 캠페인 현황 조회

1. 관리자가 캠페인 리스트 페이지 접속
2. 캠페인보기/상품별보기 뷰모드 선택
3. 기간, 캠페인명, 인플루언서, 브랜드 등 필터 적용
4. 캠페인별 인플루언서 수, 브랜드 수, 주문 수, 매출 확인

### 캠페인 생성

1. 관리자가 캠페인 제목, 기간, 설명, 썸네일 설정
2. 상품 연결 (여러 상품 선택 가능)
3. 인플루언서 배정 + 기간/수수료 유형/수수료율 개별 설정
4. 인플루언서별 상품 커스텀 커미션 설정 (선택)

### 성과 모니터링

1. 캠페인보기: 캠페인 단위로 총 주문/매출 확인
2. 상품별보기: 개별 상품의 캠페인 기여도 확인
3. 브랜드/카테고리별 필터로 특정 영역 성과 분석
