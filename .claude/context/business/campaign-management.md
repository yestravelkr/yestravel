# 캠페인 관리

## 목적

인플루언서 마케팅 캠페인을 생성하고 관리한다. 캠페인에 상품과 인플루언서를 연결하여 판매 링크를 생성하고, 매출을 추적한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 캠페인 목록 조회 | 기간/캠페인명/인플루언서/브랜드 필터링 | 관리자가 캠페인 현황을 빠르게 파악 |
| 캠페인 생성 | 기본정보 + 상품 + 인플루언서 설정 | 새 마케팅 캠페인 시작 |
| 캠페인 수정 | 기존 캠페인 정보 변경 | 진행 중인 캠페인 조건 조정 |
| 뷰 모드 전환 | 캠페인별 / 상품별 보기 토글 | 관점에 따른 데이터 분석 |
| 매출 상세 조회 | 캠페인별 매출 데이터 확인 | 캠페인 성과 분석 |

## 사용자 흐름

1. 관리자가 캠페인 목록 페이지에서 필터(기간, 캠페인명, 인플루언서, 브랜드)로 캠페인 검색
2. 캠페인보기/상품별보기 뷰 모드를 전환하여 다양한 관점으로 데이터 확인
3. 신규 캠페인 생성: 기본정보(이름, 기간) → 상품 연결 → 인플루언서 배정
4. 기존 캠페인 수정: 상세 페이지에서 편집 모드 전환 → 정보 변경 → 저장
5. 매출상세 링크를 통해 캠페인 성과 확인

## 도메인 규칙

- 캠페인은 시작일~종료일 기간을 가진다
- 캠페인에는 여러 상품과 인플루언서를 연결할 수 있다
- 인플루언서별로 맞춤 수수료와 기간을 설정할 수 있다
- 캠페인 삭제 시 확인 모달을 통해 안전하게 처리한다
