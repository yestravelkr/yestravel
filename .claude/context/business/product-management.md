# 상품 관리

## 목적

관리자가 백오피스에서 숙박(호텔) 상품을 등록하고 수정할 수 있게 한다. 상품 기본 정보, 옵션/가격, 취소 수수료 정책을 한 화면에서 관리하여 운영 효율을 높인다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 상품 생성 | 호텔 상품의 기본 정보/옵션/가격을 입력하여 등록 | 새 숙박 상품을 플랫폼에 추가 |
| 상품 수정 | 기존 상품 정보를 불러와 수정 | 가격 변경, 정보 업데이트 |
| 템플릿 가져오기 | 기존 상품 템플릿을 선택하여 폼 자동 채움 | 반복 등록 시 시간 절약 |
| 취소 수수료 설정 | 체크인 N일 전 기준으로 수수료 비율 지정 | 취소 정책을 단계별로 설정 |

## 사용자 흐름

1. 관리자가 백오피스에서 "상품 등록" 진입 → 빈 폼 표시
2. 기본 정보(상품명, 설명, 브랜드) 입력
3. 옵션 및 가격 설정
4. 취소 수수료 정책 추가 (예: 체크인 7일 전 30%, 3일 전 50%, 1일 전 100%)
5. "상품 등록" 클릭 → 상품 생성 완료
6. 수정 시: 상품 목록에서 선택 → 기존 데이터가 폼에 로드 → 수정 후 저장
