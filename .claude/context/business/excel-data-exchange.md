# 엑셀 데이터 교환

## 목적

관리자가 주문/정산 데이터를 엑셀로 내보내고, 수수료/상품옵션 데이터를 엑셀로 일괄 가져올 수 있도록 하여 대량 데이터 관리의 편의성을 제공한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 주문 내보내기 | 전체 주문 데이터를 엑셀 파일로 다운로드 | 주문 현황을 엑셀로 확인/가공 |
| 정산 내보내기 | 인플루언서/브랜드별 정산 데이터를 엑셀로 다운로드 | 정산 내역을 엑셀로 확인/공유 |
| 수수료 엑셀 처리 | 인플루언서 호텔옵션 커미션을 엑셀로 다운로드/업로드 | 대량 날짜별 커미션을 엑셀로 편리하게 설정 |
| 상품옵션 엑셀 처리 | 상품 생성 시 날짜별 옵션을 엑셀로 다운로드/업로드 | 대량 날짜/재고/가격을 엑셀로 일괄 입력 |

## 사용자 흐름

### 주문/정산 내보내기 (서버사이드)

1. 관리자가 백오피스에서 [엑셀 다운로드] 클릭
2. 서버에서 ExcelJS로 엑셀 파일 생성 → S3 업로드
3. presigned URL 반환 → 브라우저에서 자동 다운로드 (1시간 유효)

### 수수료/상품옵션 처리 (클라이언트사이드)

1. 관리자가 [엑셀 다운로드]로 현재 데이터가 담긴 템플릿 다운로드
2. 엑셀에서 데이터 수정
3. [엑셀 업로드]로 수정된 파일 업로드
4. 유효성 검증 후 미리보기 → [적용]으로 반영

### 기술 스택

- **라이브러리**: ExcelJS (xlsx 라이브러리에서 마이그레이션)
- **서버사이드**: ExcelJS → `Buffer.from(writeBuffer())` → S3 업로드
- **클라이언트사이드**: ExcelJS → `writeBuffer()` → Blob → 브라우저 다운로드
