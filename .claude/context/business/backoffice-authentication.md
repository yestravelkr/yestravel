# 백오피스 인증

## 목적

백오피스 관리자가 시스템에 안전하게 접근할 수 있도록 인증 및 권한 관리를 제공한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
|------|------|------------|
| 관리자 로그인 | 이메일 기반 인증 | 관리자가 백오피스에 접근 |
| 역할 기반 접근 제어 | ADMIN, MANAGER, GUEST 역할 구분 | 권한에 따른 기능 제한 |
| 자동 세션 유지 | 토큰 자동 갱신으로 재로그인 없이 작업 지속 | 끊김 없는 관리 작업 |
| 자동 로그아웃 | 토큰 갱신 실패 시 자동 로그아웃 | 보안을 위한 세션 만료 처리 |

## 사용자 흐름

1. 관리자가 로그인 → 액세스 토큰 발급 및 상태 저장
2. 토큰 만료 시 → 자동으로 토큰 갱신 요청 → 세션 유지
3. 갱신 실패 시 → 자동 로그아웃 → 재로그인 필요
4. 수동 로그아웃 → 상태 초기화 → 로그인 페이지 이동
