## 빠른 네비게이션

- [개발 명령어](./development-commands.md) - 개발에 필요한 모든 명령어
- [아키텍처 개요](./architecture.md) - 시스템 아키텍처 및 설계 패턴
- [새 기능 추가하기](./adding-features.md) - 새로운 기능 구현 가이드
- [데이터베이스 가이드](./database.md) - 데이터베이스 설정 및 마이그레이션 가이드

- [Docker 설정](./docker.md) - Docker 환경 설정

## Claude Code 통합

이 문서는 Claude Code와 원활하게 작동하도록 설계되었습니다. 각 파일은 Claude가 코드베이스를 이해하고 효과적으로 작업할 수 있는 상세한 정보를 포함하고 있습니다.

## 시작하기

1. [아키텍처 개요](./architecture.md)를 읽고 시스템 설계를 이해하세요
2. [개발 명령어](./development-commands.md)에서 사용 가능한 스크립트를 확인하세요
3. [Docker 설정](./docker.md)을 따라 개발 환경을 구성하세요
4. 새로운 기능을 구현할 때 [새 기능 추가하기](./adding-features.md)를 참고하세요

Claude Code 사용자를 위한 안내: 모든 문서 파일은 AI 보조 개발을 위한 명확한 컨텍스트와 가이드를 제공하도록 구성되어 있습니다.
